MessageRequest

public struct MessageRequest: JSONEncodable

A request formatted for the Conversation service.

    public init(
        input: Input,
        alternateIntents: Bool? = nil,
        context: Context? = nil,
        entities: [Entity]? = nil,
        intents: [Intent]? = nil,
        output: Output? = nil)

    Parameters

    input

    An input object that includes the input text.
    alternateIntents

    Whether to return more than one intent. Set to true to return all matching intents. For example, return all intents when the confidence is not high to allow users to choose their intent.
    context

    State information for the conversation. Include the context object from the previous response when you send multiple requests for the same conversation.
    entities

    Include the entities from a previous response when they do not need to change and to prevent Watson from trying to identify them.
    intents

    An array of name-confidence pairs for the user input. Include the intents from the request when they do not need to change so that Watson does not try to identify them.
    output

    System output. Include the output from the request when you have several requests within the same Dialog turn to pass back in the intermediate information.
