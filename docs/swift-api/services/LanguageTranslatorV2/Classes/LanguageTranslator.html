LanguageTranslator

public class LanguageTranslator

The Watson Language Translator service provides domain-specific translation utilizing Statistical Machine Translation techniques that have been perfected in our research labs over the past few decades.

  • createModel(fromBaseModelID:withGlossary:name:failure:success:)

    Create a custom language translator model by uploading a TMX glossary file.

    Depending on the size of the file, training can range from minutes for a glossary to several hours for a large parallel corpus. Glossary files must be less than 10 MB. The cumulative file size of all uploaded glossary and corpus files is limited to 250 MB.

    Declaration

    Swift

    public func createModel(
    fromBaseModelID baseModelID: String,
    withGlossary forcedGlossary: URL,
    name: String? = nil,
    failure: ((Error) -> Void)? = nil,
    success: @escaping (String) -> Void)

    Parameters

    fromBaseModelID

    Specifies the domain model that is used as the base for the training.
    withGlossary

    A TMX file with your customizations. Anything that is specified in this file completely overwrites the domain data translation. You can upload only one
    name

    The model name. Valid characters are letters, numbers, -, and _. No spaces. glossary with a file size less than 10 MB per call.
    failure

    A function executed if an error occurs.
    success

    A function executed with the modelID of the created model.
