ContentItem

public struct ContentItem: JSONEncodable

An input model for content to be analyzed by Personality Insights.

    Parameters

    content

    Content to be analyzed. Up to 20MB of content is supported.
    id

    Unique identifier for this content item.
    created

    Timestamp that identifies when this content was created, in milliseconds since midnight 1/1/1970 UTC. Required only for results about temporal behavior data.
    updated

    Timestamp that identifies when this content last updated, in milliseconds since midnight 1/1/1970 UTC. Required only for results about temporal behavior data.
    contentType

    MIME type of the content. For example, text/plain (the default) or text/html. The tags are stripped from HTML content before it is analyzed. Other MIME types are processed as-is.
    language

    Language identifier (two-letter ISO 639-1 identifier). Both English (en) and Spanish (es) input content are supported. The default is English. In all cases, regional variants are treated as their parent language; for example, en-US is interpreted as en.
    parentID

    Unique id of the parent content item. Used to identify hierarchical relationships between posts/replies, messages/replies, etc.
    reply

    Indicates whether this content item is a reply to another content item.
    forward

    Indicates whether this content item is a forwarded/copied version of another content item.
