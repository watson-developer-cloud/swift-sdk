Modality

public enum Modality: String

The nature of the related mention.

  • asserted

    A relation that is asserted as fact and which can be interpreted as such in the world in which the arguments exist.

    Declaration

    Swift

    case asserted = "ASSERTED"
  • other

    A relation that can be interpreted to hold only in a counterfactual world.

    Declaration

    Swift

    case other = "OTHER"
