RetrieveAndRank

public class RetrieveAndRank

The IBM Watson Retrieve and Rank service combines two information retrieval components into a single service. The service uses Apache Solr in conjunction with a machine learning algorithm to provide users with more relevant search results by automatically re-ranking them.
