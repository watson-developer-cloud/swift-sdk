public struct NewWord: JSONEncodable

A new word to add to the custom language model.

  • word

    A custom word from the custom model. The spelling of the word is used to train the model.

    Declaration

    Swift

    public let word: String?
  • soundsLike

    An array of pronunciations for the custom word. You can specify a maximum of five sounds-like pronunciations for each word. For pronunciation rules, refer to the following link: https://console.bluemix.net/docs/services/speech-to-text/custom.html#addWords

    Declaration

    Swift

    public let soundsLike: [String]?
  • displayAs

    The spelling of the custom word that the service uses to display the word in a transcript. If no display-as value is provided for the word, an empty string is stored here and the word is displayed as it is spelled.

    The word that is to be added to the custom model. Do not include spaces in the word, use a dash or underscore.
