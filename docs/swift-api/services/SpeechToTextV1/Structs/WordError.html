WordError

public struct WordError: JSONDecodable

An object describing problems with the custom word.

  • json

    A key-value pair that describes an error associated with the word’s definition in the format element: message, where element is the aspect of the definition that caused the problem and message describes the problem. You must correct the error before you can train the model.
