TradeoffAnalytics

public class TradeoffAnalytics

The IBM Watson Tradeoff Analytics service helps people make better choices when faces with a decision problem that includes multiple, often conflicting, goals and alternatives. By using mathematical filtering techniques to identify the top options based on different criteria, the service can help users explore the trade-offs between options to make complex decisions.
