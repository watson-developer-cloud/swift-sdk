Range

public enum Range: JSONEncodable, JSONDecodable

A range of valid values for a column.

  • dateRange

    High and low values that define the range of a DateTime column. Valid only for DateTime columns.

    Declaration

  • numericRange

    High and low Double values that define the range of a Numeric column. Valid only for Numeric columns.

    Declaration

    Swift

    An array of valid values that define the range of possible values for a Categorical column. Valid only for Categorical columns.
